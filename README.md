# Information Security Risk Assessment: A Critical Appraisal of Attention-Based View Theory
- Course: CMP9809M – Information Systems Security
- Institution: University of Lincoln
- Grade Received: 72%
- Format: Academic Report (3000 words ±10%)

# Overview
This report critically appraises the case study paper by Shaikh and Siponen (2023) titled "Information security risk assessments following cybersecurity breaches: The mediating role of top management attention to cybersecurity".

The report:
- Evaluates the Attention-Based View (ABV) theoretical model and its application to cybersecurity breach response
- Identifies merits, strengths, and limitations of the ABV model
- Proposes a new "top-down" model for Information Security Risk Assessment (ISRA) before cybersecurity breaches occur
- Supports the proposed model with theoretical foundations (Upper Echelons Theory, Behavioural Theory of the Firm, Institutional Theory) and real-world case studies (Equifax, Maersk NotPetya, GDPR)
